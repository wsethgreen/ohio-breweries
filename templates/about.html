Ohio Breweries Data

All data for this site was pulled from the Open Brewery DB public API. The data was pulled using Python's requests library. Data was organized using Python's pandas library. The brewery map was created using Python's plotly express library.

Site Deployment

This site was developed using Python's Flask library. This site was deployed via Heroku. All code for this site can be viewed at my github repository.
